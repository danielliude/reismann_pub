如何成为旅客


注册并登录后，点击“我的账户“并进入个人账户页面，并请按以下步骤操作：

控制面板->在“用户类型”项选择“旅客”并点击“保存”

如果您只是想寻找和联系服务商并且暂时不通过本网站预定，您直接用本网站搜索，不需要注册或登录。

寻找服务/服务商


完善您的个人资料

为了显示您寻找服务的诚意，让服务商更好的了解您的需求以便准确地提供更适合您的服务，Frelanvo建议您在联系服务商前先完善您的个人资料。

寻找合适的旅行服务/服务商

Frelanvo网站上有众多的旅行服务商，请您选择时认真阅读服务说明，并且货比三家。

通过阅读其他旅客的评价，您可对该服务有更多的了解。

在搜索时，点击图标或点击关注，即可将该服务保存到您的关注名单。

通过站内消息或其主页留下的联系方式，您可以联系服务商。请您介绍您的旅行计划和需求，并与服务商就细节进行沟通，以确认能得到自己满意的服务。

预定旅行服务


预定旅行服务

在服务商所提供的服务页面，您可以点击“预约”按钮对该服务进行预约。接下来您将被要求填写“服务预约协议”。填写完成并点击“提交”后，网站会将您的预约通知服务商。当服务商收到您的预约并就您提交的“服务预约协议”进行确认后，Frelanvo网站将生成协议文本，记录双方的权利与义务，并发送给您和服务商。

注意：请在填写“服务预约协议”前联系服务商并确定服务细节，因为服务商有权对您填写的“服务预约协议”进行修改，并返回给您并等待您确认。如果返回给您后，您不予确认，网站系统将删除此次预约。
