常见问题


我可以在Frelanvo上提供哪些服务？

所有与自由行相关的服务，如导游、翻译、司机、接送机。Frelanvo还包含代购服务，不管是旅客自己来去，还是代购完邮寄给顾客。此外，代定服务包括待定住宿、各类门票、休闲商务项目等。

如何寻找最适合我的服务商？

我如何知道服务商是否履行承诺？首先，通过电话、邮件等方式与服务商就所需服务进行深入沟通并达成一致。然后通过Frelanvo预定系统预定服务商。在填写预定单时，请将沟通好的服务一一列出(或请服务商通过修改订单列出)。当服务结束后，对照订单，根据所提供的服务进行付费。

Frelanvo是否对服务商进行监管？Frelanvo是一个基于公平和信任的网络平台。我们设立多项网站规章以确保服务商给出真实的信息并以履行自己的服务承诺。比如我们会核实注册用户邮箱，并要求服务商提供真实姓名。在服务结束后，我们给予顾客权利对服务商进行评分，等等。尽管如此，我们所做的一切只是为您能够更好的寻找的需要的服务。最后，还得您自行选择服务商并监督其服务情况。

服务商的星级意味着什么？如果对服务满意，旅客将给予服务商好评。星级的高低，代表着服务商收到好评的多少。更高的星级，代表之前更多旅客对其服务的肯定。在同等条件下，我们建议您选择更高星级的服务商。

在Frelanvo上预定安全吗？

我如何保证预定的服务商能按照约定与我碰面？绝大多数在frelanvo上预定的服务都会按约定执行。我们建议您在预约成功后、出发之前再次与所定的服务商确认服务，并保存好其联系方式。

如何确保我的人身财产安全？服务商在Frelanvo都会留下真实姓名存档。请您在与服务商碰面时，要求其出示有效证件，并与预订单核对，确认真实姓名。如果服务商无法出示有效证件或者核对有出入，请立即终止与其接触，并告知Frelanvo。

如果我对服务不满意怎么办？如果您认为服务商没有履行服务承诺，或者服务质量与说明有出入，请给予服务商差评。对于情况严重者，请与Frelanvo联系并申请仲裁。

服务费用如何支付？

Frelanvo对于服务费用何时以及用何种方式支付不做规定。旅客和服务商可以自行协商，如是否需要支付预付款，何时结清所有款项，以及是汇款还是现金交易。我们建议初次碰面的时候不要一次性付清，而是等服务结束后，对照预订单结算剩余款项。
